# SalvadoreX - AI-Powered Point of Sale System

Enterprise-grade POS system with advanced AI capabilities for restaurants and retail.

## Features
- Multi-tenant POS with AI capabilities
- Voice ordering with ElevenLabs
- Menu digitalization with Gemini Vision
- White-label support
- Dual database configuration
- RBAC with granular permissions
- Mexican electronic invoicing (CFDI 4.0)

## Getting Started
```bash
npm install
npm run dev
```
